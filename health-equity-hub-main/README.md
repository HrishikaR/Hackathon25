# Hackathon25
ISE'25 hackathon
This project aims to develop a unified AI-powered app providing health awareness,  mental health support, and EHR management for ASHA workers with offline access and secure data handling<img width="3613" height="81" alt="image" src="https://github.com/user-attachments/assets/9e55b621-eae2-40fa-9d59-30010e7f6c99" />
*Modular Architecture:** Separate modules for health awareness, mental health support, and EHR management to allow independent updates and scalability.
2. **AI Integration:**

   * Chatbot with **NLP for personalized, multilingual education**.
   * Sentiment analysis and federated learning for privacy-preserving mental health insights.
   * AI diagnostics and voice-to-text support for EHR data entry.
3. **Offline Support:**

   * On-device AI models and local caching to ensure usability without internet.
   * Periodic synchronization with secure cloud storage when connectivity is available.
4. **User Engagement:**

   * Gamified learning for disease awareness.
   * Mood tracking and weekly trend visualization.
   * Guided meditation timer with customizable duration.
5. **Security & Compliance:**

   * End-to-end encryption of sensitive data.
   * Federated learning for privacy.
   * Compliance with health data regulations.

---Frontend / Web & Mobile:**

  * **React** (UI development)
  * **TypeScript** (type safety)
  * **Vite** (fast build tool & dev server)
  * Tailwind CSS** (utility-first responsive design)
* **AI / ML:**

  * Open-source **LLMs** for NLP chatbot (e.g., LLaMA 2, MPT-7B-Instruct)
